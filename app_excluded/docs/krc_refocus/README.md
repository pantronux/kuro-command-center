# KRC Refocus Documentation

Kuro Research Center (KRC) is now the playground-first Kuro Playground
workspace for research.
Kuro Stack (KS) remains the daily chat surface and must not share chat history
or databases with KRC.

## Phase Index

- `00_baseline.md` - backup and baseline test record
- `01_profile_flags.md` - KRC/legacy/dev profile control plane
- `02_03_04_ui_playground.md` - UI V1 navigation, Research Console, landing page
- `05_06_knowledge_api.md` - approved Knowledge API and candidate review
- `07_bloatware_matrix.md` - daily/bloat feature deactivation matrix
- `08_scheduler_control.md` - background job gates in KRC profile
- `09_legacy_qa_deactivation.md` - optional legacy QA route deactivation
- `final_acceptance.md` - acceptance checklist and verification summary

## Operating Rule

KRC can publish approved knowledge for KS/KKG. KS can read approved knowledge
and later submit candidate knowledge when the candidate flag is enabled. KS
does not read KRC chat history, and KRC does not read KS daily history.
